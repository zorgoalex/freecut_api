# Research Drafts

Use this folder for intermediate notes from a specific branch, worktree, or hypothesis.

Suggested filename format:

```text
YYYY-MM-DD-vNN-short-hypothesis-name.md
```

Suggested draft structure:

```markdown
# VNN: Hypothesis Name

## Goal

## Setup

## Intermediate Observations

## Measurements

## Visual Findings

## Final Conclusion Candidate
```

Drafts are allowed to be incomplete. The canonical research log should receive only the final compressed conclusion after the hypothesis is tested.
